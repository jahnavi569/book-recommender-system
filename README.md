#  Book Recommendation System

## Overview

Developed a Machine Learning-powered Book Recommendation System that suggests similar books based on user preferences and historical rating patterns. The solution uses Collaborative Filtering and K-Nearest Neighbors (KNN) to generate personalized recommendations from over 1 million user-book ratings.

## Key Features

* Personalized book recommendations
* KNN-based Collaborative Filtering
* Cosine Similarity for book matching
* Interactive Streamlit Web Application
* Real-time recommendation generation
* Model persistence using Pickle

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Pickle

## Dataset

Book-Crossing Dataset

* 270K+ Books
* 278K+ Users
* 1M+ Ratings

## Results

* Built a recommendation engine using item-based collaborative filtering.
* Generated relevant book suggestions based on user interaction patterns.
* Deployed an interactive Streamlit dashboard for real-time recommendations.

## Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
